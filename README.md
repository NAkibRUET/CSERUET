# CSERUET
Node - react project

## Install
First, install NodeJs and MongoDB on the computer, then clone the repo via git:

```bash
$ git clone https://github.com/NAkibRUET/CSERUET.git
```

## Run
```bash
$ npm install
$ npm run dev
```
